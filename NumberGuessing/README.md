# Number Guessing Game (Python)

A simple terminal game. The computer picks a random number and the goal is to guess it in as few attempts as possible.

## Features
- 5 difficulty levels:
  - `easy`: 1–10  
  - `medium`: 1–100  
  - `hard`: 1–1,000  
  - `extreme`: 1–10,000  
  - `impossible`: 1–1,000,000
- Input validation (prevents crashes)
- Displays number of attempts
- Replay support

## Run
```bash
python main.py
